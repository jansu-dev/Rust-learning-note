## 变量和可变性  

 - 关键字说明：
    - immutable : 为了提供Rust的安全性与比并发性，默认的变量的值是不可以改变的;
    - mutable : 为了丰富语法完整性，使用 mutable 关键字可以实现与普通变量一样的功能；
    - demo ： 
      ```rust
        // 可变变量 x 正常输出
        fn main(){
          let mut x = 5;
          printl!("The value of is {}",   x);
          x = 6;
          printl!("The value of is {}",   x);
        }
      ```

 - 常量与变量区别：
    1. 关键字不同：不允许对常量使用 mut 关键字，应该使用 const 关键字，常量不是默认不可变，而是永远不可变  
    2. 作用域不同：常量可以在任何作用域中声明，包含全局作用域  
    3. 使用规则不同： 常量只能被设置为常量表达式，不能是函数调用的结果，或任何运行时结果  
    - demo:
      ```rust
      // Rust 常量的命名规范是使用下划线分   隔的大写字母单词  
      // 并且可以在数字字面值中插入下划线来  提升可读性
      const MAX_POINT: u32 = 100_000;
      ```

 - 隐藏shadowing：
    - shadowing 涵义：使用 let 关键字可以使得变量后赋值的变量的值被展现，之前变量的值被隐藏  
    - shadowing 与 mut区别：
      1. 实现原理不同：当再一次使用 let 关键字改变变量值时，本置是创建了一个新的变量，复用变量名字而已； 
    - demo:  
      ```rust
        let spaces = "    "
        let spaces = space.len();
      ```

 

## 数据类型  
 - rust 是静态类型语言:编译器在编译软件时使用依据数据类型和使用方法推断想逃使用的类型，一个值在使用时出现类型奇异就会出现编译报错；  
 - demo :  
    ```rust
      let guess:u32 = "42".parse().expact("Not a number!");
    ```  
 - 标量类型
   - 标量类型涵义：标量代表单独的一个值  
   - 浮点分类：整型、浮点、布尔、字符  


 - 复合数据类型
   - 符合类型涵义：可以将多个值合成一个类型  
   - 符合数据类型分类：元组、数组  


## 函数如何工作  
 - 函数命名风格(snake case)：所有字母必须小写，并使用下划线分隔单词。
 - demo：
    ```rust
     fn main(){
       another_fn();
     }
     another_fn(){
       println!("Another function");
     }
    ```
